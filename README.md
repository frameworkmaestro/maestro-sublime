# maestro-sublime
Package com funcionalidades para o maestro

# Como Instalar?
Faça o download do projeto zip e descompacte no diretório do <sublime>\Packages

No windows, fica em: C:\Users\<usuario>\AppData\Roaming\Sublime Text 2\Packages\
